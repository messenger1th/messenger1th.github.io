区间dp

通常来说，解决区间dp的步骤是

  • 找到依赖关系， 这一步决定了dp的维度和循环数
  • 初始化
  • 枚举长度length $\to$ n，从小区间j转移到大区间

区间dp是自底向上的，有些题目是可以从自顶向下转换成区间dp的。

记忆化递归是自顶向下的。
